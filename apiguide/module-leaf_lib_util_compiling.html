leaf/lib/util/compiling

Utility functions for compile source code.

Methods

<static> compileHandlebars(source, options) → {function}

Compile handlebars template string.

Parameters:
Name Type Description
source string

Template source string.
options object

Optional settings.

Properties
Name Type Argument Description
helpers object <optional>

Handlebars helper functions to register.
See:
  • http://handlebarsjs.com/
Returns:

Handlebars template function.

Type
function
Example
var tmpl = compileHandlebars('Here are {{toLowercase name}}.', {
         helpers: {
             toLowercase: function(str){ return str.toLowerCase(); }
         }
    });
    console.log(tmpl({name: 'Red Apples'}));
         // -> Here are red apples.
